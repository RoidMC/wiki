Section titled “🚒 许可证”

UniX SDK采用双重许可：

  1. Mozilla Public License 2.0 (MPL-2.0)

    • 允许将SDK与专有代码结合使用
    • 对SDK的修改必须在相同许可下发布
    • 完整许可证文本请见LICENSE文件

  2. 归属要求

    • 使用UniX SDK的应用必须显示”Powered by UniX SDK”
    • 具体要求请见ATTRIBUTION文件

🚀 快速开始

Section titled “🚀 快速开始”

在Server目录中添加UniX SDK

Section titled “在Server目录中添加UniX SDK”

下载SDK后在根目录下创建Public并在内创建sdk文件夹，将SDK文件放入其中

您也可以在main.lua中修改SDK路径，同时请注意修改Require的路径
